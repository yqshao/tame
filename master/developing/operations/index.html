Operations

TAME takes advantage of established libraries for heavy numerical tasks. Currently, the computations are performed with the Numpy library. Other computation backends might be implemented in hte future.

documentation in construction

The documentation now contains an incomplete list of available operations in TAME, with very brief introductions of what they do, a more detailed API documentation should be available later.

Time-dependent Operations

Below lists operations that are time-dependent, which means the value of each frame depend on previous frames.

function operation
tm.op.unwrap unwraps the coordinate 1

Basic Functions

The below functions are basic mathematical operations, those functions are also defined as python operators between tm.FrameArrays.

function syntax operation
tm.plus a+b addition
tm.minus a-b subtraction
tm.product a*b multiplication

  1. TAME unwraps the trajectory using the minimal displacement vector at each time step, to avoid possible artifact when unwrapping constant-pressure simulations, which is discussed by von Bülow et al. in detail.2 ↩

  2. Sören von Bülow, Jakob Tómas Bullerjahn, and Gerhard Hummer. Systematic errors in diffusion coefficients from long-time molecular dynamics simulations at constant pressure. J. Chem. Phys., 153:021101, 2020. doi:10.1063/5.0008316. ↩
